## Tic-Tac-Toe

This is a tic-tac-toe game that can be played on the terminal.

This project was made as a fun way to practice string formatting and practice TDD 
with a bootstrapped test framework.

To play, run:
```
 $ ./play.sh
```

To run all tests:
```
 $ ./run_all_tests.sh
```


Enjoy!
